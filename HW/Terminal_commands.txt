
pwd - показать папку, где находимся
cd ../ - выйти на уровень выше
mkdir group_32 - создать папку group_32

cd group_32 - зайти в папку
mkdir d1 d2 d3 - создать несколько папок d1 d2 d3 в одной папке 
ls - посмотреть содержимое текущей папки
По-умолчанию ls сортирует файлы по алфавиту

ls -a посмотреть содержимое текущей папки, позволяет увидеть скрытые файлы
ls -l посмотреть содержимое текущей папки, выводится подробная информация об объектах
ls -la посмотреть содержимое текущей папки, показывает скрытые файлы и подробную информацию по каждой папке/каждому файлу
cd d1 - зайти в папку d1
cd ../.. - выйти на два уровня выше 
touch f1.txt - создать файл 
 
cat >> f2.txt - создать файл и зайти в режим редактирования 
Ctrl + c выйти из режима редактирования

cat f2.txt - посмотреть сожержание файла

cp f2.txt f3.txt скопировать сожержимое файла f2.txt в созданный файл f3.txt
cp f1.txt ../d3/f1.txt - в папку d3 копируется файл f1.txt
cd ../d3 перейти из папки d1 в d3
cp -r d1/d11 d3 - скопировать папку d11 из d1 в d3  cat

mv f1.txt ../d2/f1.txt переместить файл f1.txt из текущей папки в папку d2.
cd ../d2 перейти из текущей папки в d2
mv f1.txt f3.txt переименовать файл f1.txt в f3.txt

vim f2.txt - редактирование файла f2.txt
клавиша i - включить режим редактирования
Esc - выход из режима редактирования
:wq - выход из редактора vim с сохранением изменений
w - write (save) and q - quit
:q! - выход из редактора vim без сохранения изменений

cat >> f1.txt - запустить режим редактирования, новые строчки пишутся в конец файла

cat f1.txt > f2.txt заместить содержание файла f2.txt данными из файла f1.txt

rm f2.txt - удалить файл 
rm -r d1 - удалить папку

less [OPTIONS] filename
[OPTIONS] 
less -N filename - показать номера строк
less -X filename - оставить содержимое файла на экране после выхода из less
less -pPATTERN filename откроет файл на странице, на которой первый раз встречается PATTERN
less +F filename - посмотреть содержимое файла в режиме реального времени. 
После просмотра в реальном времени выйти из less и вернуться в командную строку: Ctrl + c, :q

Команд для навигации по содержимому файла
клавиша f - переход на следующую страницу
клавиша b - переход на предыдущую страницу
клавиша q - выйти из less и вернуться в командную строку
клавиша p - перейти к началу файла
/pattern - Поиск вперед для соответствия шаблонам.
?pattern - Поиск назад для соответствия шаблонам.

Можно открыть несколько файлов 
less f1.txt f2.txt
:n - показать следующий файл
:p - показать предцдущий файл

find <где искать> <опции>

<где искать> — путь к корневому каталогу, откуда начинать поиск. 
Например, find /home/user — искать в соответствующем каталоге. 
Для текущего каталога нужно использовать точку «.»

<опции> — набор правил, по которым выполнять поиск.
-name - Поиск по имени
-iname - Регистронезависимый поиск по имени.
-type f - поиск файлов
-type d - поиск папок
find . -type f -name "*.php" find all php files in a directory

Создать скрипт:
1. Создайть пустой файл с использованием команды touch:
touch filename
Первая строка файла будет такой #!/bin/bash
2. Сделать файл исполняемым в Linux:
chmod +x ./filename

chmod <категория> <действие> <флаг> адрес_файла

<категория> - флаги могут устанавливаться для трех категорий: владельца файла, группы файла и всех остальных пользователей. В команде они указываются символами u (user) g (group) o (other) соответственно.
<действие> - может быть + (плюс), что будет значить установить флаг или - (минус) снять флаг.
<флаг> - один из доступных флагов - r (чтение), w (запись), x (выполнение).

3. Обновить скрипт:
#!/bin/bash
echo "The current directory is:"
pwd
echo "The user logged in is:"
whoamicat

4. проверить, что скрипт выполняется
./filename
В терминале будет информация о текущей дериктории и о пользователе

5. Добавить в скрипт нужные команды
6. Запустить выполнение скрипта в терминале ./filename
